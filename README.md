# 🎓 StudentHub

StudentHub is a modern all-in-one student productivity dashboard designed to help students stay organized, productive, and focused throughout their academic journey. It combines essential study tools into a single, intuitive platform, eliminating the need to switch between multiple applications.

## ✨ Features

* 📚 Assignment Manager
* ✅ To-Do List
* 📅 Study Planner
* ⏱️ Pomodoro Timer
* 📊 Attendance Calculator
* 🎓 GPA Calculator
* 🤖 AI Study Assistant (Gemini API)
* 🌤️ Live Weather Updates
* 💬 Motivational Quotes
* 🕒 Live Clock & Personalized Greeting
* 🌙 Dark/Light Theme
* 💾 Automatic Local Storage Support

## 🛠️ Built With

## ▶️ Running Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/studenthub.git
   ```

2. Navigate to the project directory:

   ```bash
   cd studenthub
   ```

3. Open the project in Visual Studio Code.

4. Install the **Live Server** extension (if you haven't already).

5. Right-click `index.html` and select **Open with Live Server**.
